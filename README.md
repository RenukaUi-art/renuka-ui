# Task Management Web Application

## Description
A web-based task management application that helps teams organize, assign, and track daily tasks efficiently.

## Features
- Responsive UI built with React.js
- Reusable component-based architecture
- REST API integration for real-time updates
- Form validations and error handling

## Technologies Used
- React.js
- JavaScript
- HTML5
- CSS3
- Bootstrap
- REST APIs
- Node.js

# Tele Directory Web Application
## Description
A web-based employee tele-directory application designed to quickly search, filter, and view employee contact details within an organization. The application provides a clean and responsive user interface with dynamic data rendering for an improved user experience.

## Features
- Employee search and filter functionality
- Dynamic data rendering using JSP and REST APIs
- Responsive UI for desktop and mobile devices
- Modular and reusable UI components
- Clean and maintainable code structure
