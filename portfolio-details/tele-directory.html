Portfolio -1: Internal Phone Directory Application

My Role: UI Developer

Project Type: Internal Company Tool

Goal: A secure, centralized communication hub to streamline internal connectivity for all employees.

Description

I designed an Internal Phone Directory Application to improve employee communication within a large organization. The tool offers a secure, centralized directory with integrated calling via Exotel, caller ID masking, and logging for transparency. My focus was on a privacy-first interface balancing access and protection. This project sharpened my UI development skills for internal tools in corporate environments.

Challenges

Creating a secure and user-friendly platform for employee contact management in a large organization with built-in calling features and comprehensive logging mechanisms.

Key Solutions & Features
  • Security-First Design: Protected employee data with access controls and detailed logging.
  • Centralized Directory: Complete contact information for all employees in one place.
  • Integrated Exotel Calling: Direct calls with features like caller ID masking and call recording.
  • Interaction Logging: Detailed logs for both calls and profile views to ensure transparency.
