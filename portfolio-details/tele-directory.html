Project Overview

  • Project Title: Internal Phone Directory Application
  • My Role: UI/UX Designer
  • Type: Internal company tool for employee use
  • Goal: To create a secure, centralized, and feature-rich communication hub for employees

The Challenge

In a large organization, managing employee contact info and ensuring seamless internal communication is complex. The task was to design a secure, user-friendly platform that offers directory access, calling features, and detailed interaction logs for accountability.

My Solution: A Secure & Integrated Communication Hub

  • Centralized Employee Information: Organized all employee contact details in one place.
  • Integrated Calling (Exotel): Enabled direct calling from the app using Exotel API.
  • Advanced Call Features: Caller ID hiding and call recording functionality.
  • Comprehensive Logging: Tracked profile views and call logs for transparency.
  • Security-First Design: Prioritized protection of sensitive data and communication records.
