📞 Tele Directory Web Application

👨‍💻 My Role: Software Developer (UI Development & API Integration)

🎯 Goal: Developed an internal tele-directory system to provide a centralized platform for accessing employee contact details and improving communication across the organization.

⚙️ My Contribution: Built responsive UI screens using HTML, CSS, JavaScript, Bootstrap, and JSP. Integrated APIs to fetch employee data dynamically and display it efficiently. Worked with backend teams to ensure secure data handling and performance.

🔐 Focus: Security, usability, and scalability for enterprise-level usage.

🔥 The Challenge

Managing employee contact information manually or across multiple systems led to inefficiencies, lack of accessibility, and communication delays.

💡 Solution
  • Centralized Directory: All employee contacts in one place
  • Advanced Search: Quick filtering by department, role
  • Real-Time Data: Dynamic API-driven content
